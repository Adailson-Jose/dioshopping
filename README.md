## Instruções para rodar a aplicação
## Frontend

```bash
  $ npm install
  $ npm start
```

##  Backend 

Clone o repositório disponível em [sportshop-api](https://github.com/volinha/sportshop-api).

Instale e rode o servidor

```bash
  $ npm install
  $ npm run dev
```
